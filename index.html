SMART NURSE CALL SYSTEM BERBASIS MICRO-ROS

Introduction

Background: The Smart Nurse Call System is a highly efficient communication tool designed to enhance the interaction between patients and healthcare providers.

Project Description: Utilizing Micro-ROS, this system aims to provide real-time updates and alerts to nursing staff.

Objectives: Improve response times, enhance patient monitoring, and streamline communication processes.
